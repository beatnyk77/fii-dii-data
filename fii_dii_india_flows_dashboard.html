📑 NSDL Fortnightly Reports 🏛️ BSE Sector Classification
ℹ️
How is this data calculated? SEBI mandates all Foreign Portfolio Investors (FPIs) to hold securities via Indian depositories (NSDL / CDSL). Every fortnight, depositories publish sector-wise Assets Under Custody (AUC). The % of AUM shown is each sector's share of the total FPI equity portfolio in India. Sectors are classified using BSE's Common Industry Classification. Flows show net FPI inflows/outflows per sector over the last 15-day reporting window and the rolling 12-month period.

Dashboard User Manual & Methodology
Complete documentation for all 7 tabs: Live NSE, FPI Macro, Sectors, Deep Dive, Databases, F&O Derivatives, and this Docs section. Covers data sources, formulas, and how to read every indicator.

⚡ 1. Live NSE (Home Dashboard)

The default landing view. Shows the latest extracted session with real-time institutional flow data from NSE TRDREQ reports.

Flow Strength Meter Visual split showing FII aggression vs DII support percentage. Calculated as: FII_NET / (|FII_NET| + |DII_NET|) × 100
Streak & Velocity Tracking Monitors consecutive buying/selling days for FIIs/DIIs with aggregated capital velocity over the streak period.
45-Day Concentration Heatmaps GitHub-style daily block rendering. Darker green = buys > ₹5,000 Cr. Darker red = sells > ₹5,000 Cr. Reveals FII sell-off depth and DII absorption density.
Quick Stats Strip 5-Year cumulative flows, SIP monthly run-rate (₹26,500 Cr), and FII NSE500 ownership percentage (16.1%, down from 28% peak).

📊 2. FPI Macro (Historical Flow Trends)

Macro-level historical trajectories across multiple timeframes. Data sourced from NSDL FPI investment reports dating back to 2005.

  • FPI Daily Trends: Latest registered equity, debt, and hybrid investment snapshots with summary cards showing net flows in crores.
  • Institutional Flow Canvas: Grand Chart.js canvas plotting FII (red) and DII (green) net flows. Switch between Daily, Weekly, Monthly, and Annual resolutions.
  • Quarterly FPI Breakdown: Quarter-over-quarter equity and debt allocations from 2015 onwards.
  • Yearly & Monthly Heatmap: Multi-year monthly data (2005–2026) showing exact FPI investment patterns across all months.

🌐 3. Sectors (NSDL Fortnightly FPI Allocation)

Tracks actual equity portfolios held by Foreign Portfolio Investors inside Indian depositories via NSDL/CDSL. Reported fortnightly by SEBI.

  • Sector Cards: Screener-style cards showing AUM %, FII Ownership %, and 24-fortnight sparkline charts with gradient area fills. Color-coded green (inflow) / red (outflow).
  • FPI Sector Flow Trend: Interactive 8-sector comparative line chart. Hover over any sector to highlight its trend line while fading others to 20% opacity.
  • Scoreboard: Top 8 inflows and outflows ranked by cumulative equity net flow with proportional progress bars.
  • Sort Modes: Total AUM, Fortnight Δ, 1-Year Net Flow, or Alphabetical ordering.

🤿 4. Deep Dive (Advanced Analytics)

Advanced datasets bridging macroeconomic demographics and debt discipline thresholds. Three independent modules.

  • Trade-Wise Equity Flow Engine: Granular monthly flow tracking mapped to the Nifty 500 via ISIN cross-referencing. Shows exactly which stocks and sectors foreign funds are targeting each month via the NSE's trade-wise data.
  • Country-wise AUC (Donut Chart): Illustrates FPI capital sources — leading with USA, Singapore, Luxembourg, and Mauritius. Data from NSDL country-wise Assets Under Custody reports.
  • ODI / Participatory Notes: Offshore Derivative Instrument tracking — shows how much foreign capital enters India via P-Notes rather than direct FPI registration.
  • Debt Utilisation: Monitors FPI debt investment limits vs actual utilisation — a critical indicator for bond market sentiment.

🗄️ 5. Databases & Matrices

Complete institutional flow archive with Bloomberg-terminal grade tabular rendering.

  • Daily (Last 15): Raw end-of-day gross buy, sell, and net. Click "FII Bloodbath" or "DII Absorption" to rapidly filter anomaly days. Includes magnitude bars for visual scanning.
  • Weekly, Monthly & Annual: Aggregated capital flows grouped by timeframe with Nifty correlation, trend signals, and Domestic Multiplier ratios.
  • CSV Export: Custom FROM/TO date filtering with instant CSV download for Excel modeling.

🎲 6. F&O Derivatives Positioning

Participant-wise Open Interest from the NSE, fetched live via the /api/participant-wise-open-interest endpoint. Updated automatically on each sync.

  • Index Futures Ratio: FII Long vs Short contracts. A heavily short position implies bearish hedging on Nifty/Bank Nifty.
  • Options Analysis: Call Long = Bullish, Put Long = Bearish. Extreme put writing correlates with market corrections.
  • Sentiment Engine: Heuristic algorithm weighing net futures, calls, and opposing put positions. Outputs: Highly Bullish → Highly Bearish.
  • Historical Positioning Chart: 20-period trajectory chart for FII Futures, Calls, Puts, and DII Futures.
  • Live Data Pipeline: Uses a 3-tier proxy system: Express server → corsproxy.io → allorigins.win to fetch NSE OI data reliably.

🏗️ 7. Data Sources & Architecture

The dashboard pulls data from multiple official government and exchange sources.

NSE TRDREQ API Daily FII/DII cash market buy/sell/net data. The primary source for the Live NSE home tab.
NSDL FPI Reports Fortnightly sector allocation, daily FPI investment, monthly/yearly aggregates, country-wise AUC, debt utilisation, and ODI/PN data.
NSE Participant OI API F&O participant-wise Open Interest for FII/DII index futures, stock futures, and index options positioning.
Serverless Fallbacks All data endpoints have JSON fallbacks in /data/ allowing the dashboard to work as a static site without the Express server.
